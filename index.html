‚ö° Quick Start Installation

Get up and running in minutes with our streamlined installation process:

1. Install OpenClaw

npm install -g openclaw@latest

Or use pnpm:

pnpm add -g openclaw@latest

2. Run the Onboarding Wizard

openclaw onboard --install-daemon

The wizard guides you through setup, workspace configuration, and channel connections.

3. Start the Gateway

openclaw gateway --port 18789 --verbose

4. Send Your First Message

openclaw agent --message "Hello, OpenClaw!" --thinking high
