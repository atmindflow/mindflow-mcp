Discord Components v2

Interactive agent prompts with buttons, selects, modals and attachment-backed file blocks

Rich Interaction UI

Build sophisticated user experiences with Discord's component system. Buttons, select menus, and modals let users interact with your agents in intuitive ways—no more plain text commands.

Event-Driven Architecture

Components trigger events that your agent receives in real-time. Handle button clicks, dropdown selections, and modal submissions with full context and state preservation.

Attachment-Backed Blocks

Present file content directly in Discord with attachment-backed file blocks. Users can view, download, or interact with files seamlessly within the conversation flow.

How It Works

1

Agent Sends Component

Agent creates a message with interactive components (buttons, selects, modals)

→
2

User Interacts

User clicks button, selects option, or submits modal in Discord

→
3

Agent Receives Event

Agent gets interaction event with full context and user choice

→
4

Execute Action

Agent processes interaction and responds or performs action

Quick Start Configuration

Enable Discord Components v2 in your OpenClaw setup:

channels:
  discord:
    componentsVersion: 2
    enableEmbeds: true

agents:
  defaults:
    subagents:
      maxSpawnDepth: 2
      maxChildrenPerAgent: 5
