WebGL/Animation notes

This page hero renders a single full-screen triangle in WebGL2. The fragment shader raymarches a Signed Distance Field (SDF) scene (deformed sphere core + tilted torus + satellite) with soft shadows, lightweight AO, Fresnel highlights, and distance fog. We avoid MSAA and post-blur; instead, we use an emissive approximation and adaptive step counts ("Boost quality" toggle) for a stable frame time. Resolution scales with DPR, capped for dense displays. Reduced-motion is respected by pausing animation by default when users have it enabled.
