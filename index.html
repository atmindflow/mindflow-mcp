Quick Installation

Get started in minutes with npm, pnpm, or bun

# Install globally
npm install -g openclaw@latest

# Run onboarding wizard
openclaw onboard --install-daemon

# Start the gateway
openclaw gateway --port 18789 --verbose

How It Works

1. Local Gateway

The Gateway runs on your machine (ws://127.0.0.1:18789) as a control plane managing all channels, tools, and agent interactions.

2. Multi-Channel Integration

Connect your preferred messaging platforms - WhatsApp, Telegram, Slack, Discord, and more - all routed through the gateway.

3. Pi Agent Runtime

Powered by Pi agent in RPC mode with tool streaming and block streaming for responsive, intelligent interactions.

4. Optional Companion Apps

Enhance with macOS menu bar app, iOS/Android nodes, CLI tools, and WebChat UI for complete control.
