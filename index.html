What’s new in 1.8.0

Exact release bullets are preserved verbatim. Each item includes a short explanation of impact.

Changed

  • 🔐️(agents) uninstall pip from the agents image Reduces attack surface and prevents accidental runtime package installs inside the container. Builds become more reproducible and least-privilege by default.
  • 🔐️(summary) switch to Alpine base image Alpine images are typically smaller, which can reduce CVE exposure and speed up pulls/rollouts while keeping the runtime focused on essentials.
  • 🔐️(backend) uninstall pip in the production image Hardens production by removing package managers that are unnecessary at runtime, limiting tampering paths if a container is ever compromised.

Fixed

  • 🔐️(agents) upgrade OpenSSL to address CVE-2025-15467 Improves cryptographic safety for agent-side operations by incorporating an OpenSSL update that mitigates a known vulnerability.
  • 📌(agents) pin protobuf to 6.33.5 to fix CVE-2026-0994 Locks protobuf to a vetted version to avoid vulnerable ranges—reducing supply-chain risk and keeping serialization stable across deployments.
