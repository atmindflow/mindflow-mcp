WebGL Background: Technical Overview

This page uses a lightweight WebGL1 renderer to create a subtle animated backdrop that evokes streaming traffic and layered wavefronts‚Äîmirroring OpenClaw‚Äôs focus on real‚Äëtime agent streams and secure network flows.

1. Context & Compositing

The script requests a WebGL1 context with alpha enabled and premultiplied alpha disabled. This allows our fragment shader to output low‚Äëopacity colors that blend beneath the main content without washing out text or UI elements.

2. Full‚ÄëScreen Quad Geometry

A single vertex buffer describes two triangles that cover the clip space from ‚àí1 to +1 in both axes. Once uploaded, this buffer is reused for every frame, making per‚Äëframe CPU overhead negligible.

3. Procedural Fragment Shader

The fragment shader computes UV coordinates from gl_FragCoord and feeds them through a combination of sine/cosine functions over time. The output is a layered gradient that slowly drifts across the screen.

4. Time & Resolution Uniforms

Two uniforms drive animation and scaling: u_time (elapsed seconds) and u_resolution (canvas width/height). Resizes update both the canvas backing store and the WebGL viewport to maintain aspect‚Äëcorrect visuals.

5. Animation & Power Efficiency

Rendering uses requestAnimationFrame. The loop automatically pauses when the document is hidden and disables entirely for users who set prefers-reduced-motion: reduce, honoring both battery and accessibility requirements.

6. High‚ÄëDPI Support

The canvas size is multiplied by window.devicePixelRatio, while CSS dimensions remain in logical pixels. The viewport is set to the physical resolution to prevent blur on Retina and other high‚Äëdensity displays.

Core Fragment Shader Logic

// Normalize coordinates to [0,1]
vec2 uv = gl_FragCoord.xy / u_resolution;

// Center and scale
vec2 p = (uv - 0.5) * 2.0;

// Layered wave fields
float wave1 = sin(p.x * 3.0 + u_time * 0.5);
float wave2 = cos(p.y * 4.0 + u_time * 0.3);
float wave3 = sin((p.x + p.y) * 2.0 - u_time * 0.2);

float combined = (wave1 + wave2 + wave3) / 3.0;

vec3 baseColor   = vec3(0.02, 0.04, 0.10);   // deep background
vec3 accentColor = vec3(0.14, 0.56, 0.96);   // OpenClaw-inspired accent

vec3 color = mix(baseColor, accentColor, 0.5 + 0.5 * combined);

gl_FragColor = vec4(color, 0.18); // low alpha for subtle overlay

The combination of three phase‚Äëshifted waves produces slow, organic motion that feels like overlapping network traces without distracting from content.
