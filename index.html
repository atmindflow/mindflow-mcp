Release tour

Reliability everywhere

Context-aware tool budgets, subagent overflow guards, stable reply threading across streams, and channel-specific fixes keep long-running conversations predictable.

  • Auto-page tools.read without breaking responses
  • Subagent truncation recovery for long tool results
  • Thread-sticky replies for better streaming continuity

Security upgrades

Exec preflight guards against env-var injection, strict include resolution, user-only transcripts, and safer config merges reduce accidental exposure.

  • OC-09 patch set
  • $include confinement / path hardening
  • 0600 transcript permissions

Cron at scale

Deterministic staggering, per-job webhook delivery, usage telemetry, and exact-timing overrides for precise automation and load smoothing.

Voice + Calls

Pre-cached greetings, optional stale-call reaper, and race-free turn handling for robust real-time conversational systems.
