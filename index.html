How It Works

OpenClaw orchestrates LLM-powered agents across channels (Discord, Slack, Telegram, Web, CLI) with a gateway, sandboxed skills, scheduling (cron), memory, and rich UX surfaces. The 2026.2.15 release upgrades Discord Components to v2 for interactive prompts, enables nested sub-agents with depth control, and opens new plugin hooks to observe LLM inputs/outputs for better governance.
