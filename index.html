Step 1: Install

Install OpenClaw globally with npm, pnpm, or bun. Requires Node ‚â•22.

Step 2: Onboard

Run the guided wizard to set up your gateway, workspace, channels, and skills.

Step 3: Start Gateway

Launch the Gateway control plane on your local machine or remote server.
