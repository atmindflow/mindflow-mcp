What's new in v2026.2.17

  • Anthropic 1M context (opt-in): set params.context1m: true to send the anthropic-beta: context-1m-2025-08-07 header for Opus/Sonnet.
  • Anthropic Sonnet 4.6: adds anthropic/claude-sonnet-4-6 with forward-compat fallback when catalogs lag.
  • Subagents: new /subagents spawn command for deterministic subagent activation.
  • Slack: true single-message streaming via chat.startStream/appendStream/stopStream (fallback when streaming fails).
  • Telegram: inline button style support (primary|success|danger) plus optional reaction notifications.
  • iOS: Share Extension forwards URL/text/images to gateway agent.request; Talk adds Background Listening and a Voice Directive Hint toggle.
  • Cron: safer per-job webhook delivery and better schedule controls (--stagger, --exact).
