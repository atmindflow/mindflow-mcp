WebGL Background: Technical Implementation

The animated background is a lightweight, zero-dependency WebGL1 implementation. It renders a full-screen quad driven by a custom fragment shader with time and resolution uniforms. Animation automatically respects prefers-reduced-motion user preferences for accessibility.

1. Context Initialization

We acquire a WebGL1 rendering context with alpha compositing enabled. This allows the shader output to blend seamlessly beneath the content layer, creating a subtle depth effect without visual interference.

2. Static Geometry Buffer

A single vertex buffer contains six vertices (two triangles) forming a quad in normalized device coordinates (-1 to 1). This static geometry is uploaded once and reused every frame, eliminating per-frame CPU-GPU transfer overhead.

3. Shader Pipeline

The vertex shader performs a simple pass-through, forwarding positions directly to gl_Position. The fragment shader computes a procedural gradient using layered sine and cosine waves, outputting low-alpha colors for an ethereal glow effect.

4. Uniform Updates

Two uniforms drive the animation: u_time (elapsed seconds) updates each frame to animate the waves, while u_resolution (canvas dimensions) updates on resize to maintain correct aspect ratios and prevent distortion.

5. Animation Loop

Rendering uses requestAnimationFrame for optimal performance and battery efficiency. The loop automatically pauses when the document is hidden (tab switching) and disables entirely for users with prefers-reduced-motion: reduce.

6. High-DPI Handling

The canvas backing store scales with window.devicePixelRatio to ensure sharp rendering on Retina and high-DPI displays. The WebGL viewport is configured to match physical pixel dimensions, preventing blur on modern screens.

// Time-based wave animation
float wave1 = sin(uv.x * 3.0 + u_time * 0.5);
float wave2 = cos(uv.y * 4.0 + u_time * 0.3);
float combined = (wave1 + wave2) * 0.5;

// Low-alpha gradient for subtle effect
vec3 color = mix(baseColor, accentColor, combined);
