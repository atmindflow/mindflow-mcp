Quick Installation

Get started with OpenClaw in just a few commands:

# Install OpenClaw globally
npm install -g openclaw@latest

# Run the onboarding wizard
openclaw onboard --install-daemon

# Start the Gateway
openclaw gateway --port 18789 --verbose
